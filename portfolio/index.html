Biomedical Engineering Projects

REAL TIMEHEART DISEASE DETECTION

This project combines heart sensors, electrode leads, and an ESP32 board to create a real-time heart disease detection system. By capturing and analyzing electrical signals from the heart, it swiftly identifies abnormalities associated with heart disease. This innovative setup promises timely alerts and insights into cardiac health, showcasing the potential of merging hardware and software for medical monitoring solutions.


AIR QUALITY MONITORING SYSTEM

The Arduino-based air quality monitoring system offers a practical solution for tracking air pollution levels. By incorporating sensors like MQ series and DHT series, it gathers data on pollutants, temperature, and humidity in real-time. This system, with Arduino's processing power, can display information on screens or send alerts to users. Its compact design and cost-effectiveness make it suitable for various settings, aiding in the proactive management of air quality concerns.


Web Development Projects

BMI CALCULATOR

This project is a simple BMI (Body Mass Index) calculator built using HTML, CSS, and JavaScript. It takes input from the user such as age, height, weight, and gender, and then calculates the BMI based on these inputs. The BMI value is then used to determine the user's weight status, whether they are underweight, healthy, overweight, obese, or extremely obese. It includes validation to ensure all fields are filled before calculating the BMI. Additionally, it features a modal to display error messages if any fields are left blank. This project demonstrates basic front-end development skills and can be a valuable addition to a portfolio showcasing web development proficiency.


HEART SYNC

This Flask web application serves as a heart disease prediction tool. It utilizes a Logistic Regression model trained on heart disease data to predict whether a person is likely to have heart disease based on various medical attributes provided by the user. The application offers a simple and intuitive interface for users to input their data and receive a prediction. Additionally, it includes pages for about information, user signup, and a map feature, demonstrating versatility in web development skills. This project showcases proficiency in Python, Flask, HTML, CSS, and integrating machine learning models into web applications, making it a valuable addition to a portfolio highlighting full-stack development capabilities and data science skills.


SKIN CANCER DETECTION

This skin cancer detection project merges machine learning with web development, offering a simple tool for early skin cancer identification. Users upload images of skin lesions, processed by a pre-trained model, which predicts the likelihood of cancer. This web application enhances accessibility to skin health assessment, potentially aiding in early detection. The project showcases expertise in both machine learning and web development, serving as a succinct demonstration of AI's application in healthcare through intuitive web interfaces.
